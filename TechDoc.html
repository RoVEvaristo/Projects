Introduction to HTML

HTML stands for HyperText Markup Language. It is the standard language for creating web pages.

Every HTML document starts with a <!DOCTYPE html> declaration.

  • HTML structures web content
  • HTML elements are building blocks
HTML Elements

Elements are defined by tags, such as <p> for paragraphs.

They can have attributes like id or class to identify them.

  • Heading tags: <h1> to <h6>
  • Lists: <ul> or <ol>
CSS Styling

CSS (Cascading Style Sheets) controls the presentation of HTML elements.

Use classes and IDs to apply styles selectively, for example:

.example { color: red; }
  • Fonts and colors
  • Layout and spacing
JavaScript Basics

JavaScript adds interactivity to web pages.

Variables can be defined using let, const, or var.

  • Functions perform tasks
  • Events respond to user actions
DOM Manipulation

The Document Object Model (DOM) represents the page structure.

JavaScript can be used to modify elements dynamically:

document.getElementById('demo').innerText = 'Hello';
  • Select elements
  • Update content
